# imgbb_uploader

## Installation for Mac

```
brew install poppler
```

```
pip3 install Pillow imgbbpy pyheif pdf2image
```

## ImgBB API Key

[Get the API key from ImgBB](https://api.imgbb.com) and set to it `IMGBB_API_KEY` in `~/.zshrc`

```
vim ~/.zshrc
```

```
IMGBB_API_KEY=xxxxyyyyyzzzzz
```
